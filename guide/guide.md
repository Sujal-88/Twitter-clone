# How to use the Tailwind CSS
## Step-1:
```
npm init -y
```
## Step-2:
```
npm install tailwindcss @tailwindcss/cli
```
## Step-3:
```
npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch
```
## Step-4:
```
npx @tailwindcss/cli -i ./css/input.css -o ./css/output.css --watch
```
## Step-5: Terminate it once
## Step-6:
```
npx tailwindcss-cli@latest init
```
